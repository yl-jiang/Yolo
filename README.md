# YoloV1

## 训练
* 直接运行'train.py'文件

## 测试
* 下载checkpoint文件，并将下载的文件放在'YOLOV1/checkpoint/'文件夹下（需要自己新建一个名为checkpoint的文件夹）
* ‘test_imgs’文件夹中添加想要测试的图片
* 运行'test.py'脚本
* 目标检测结果保存在'Visualization/result'文件夹下
